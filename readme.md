
MiniBuild命令行参数列表：

  -s, --solution         Required. 解决方案文件名（含路径）
  
  -t, --target           (Default: Build) 生成目标：compile/rebuild/clean
  
  -c, --configuration    (Default: debug) 配置类型：debug/release
  
  --silent               (Default: False) 以静默模式启动（执行后立即退出）
  
  --setup                (Default: False) 安装*.sln文件的右键菜单
